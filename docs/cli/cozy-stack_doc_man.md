## cozy-stack doc man

Print the manpages of cozy-stack

### Synopsis


Print the manual pages for using cozy-stack in command-line

```
cozy-stack doc man [directory]
```

### Examples

```
$ mkdir -p ~/share/man
$ export MANPATH=~/share/man:$MANPATH
$ cozy-stack doc man ~/share/man
$ man cozy-stack
```
